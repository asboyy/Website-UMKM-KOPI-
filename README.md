
# Talafu Coffee - Website UMKM Kopi Lokal Medan ☕

**Talafu Coffee** adalah sebuah proyek *landing page* responsif yang dirancang untuk mempromosikan produk kopi lokal dari UMKM di Medan. Website ini bertujuan untuk memberikan pengalaman visual yang menarik bagi pelanggan sekaligus memudahkan mereka dalam menjelajahi menu dan lokasi kedai.

## 🚀 Fitur Utama

* **Desain Responsif:** Tampilan yang optimal di berbagai perangkat (Mobile, Tablet, dan Desktop).
* **Navigation Menu:** Akses cepat ke bagian "Tentang", "Menu", dan "Lokasi".
* **Hero Section:** Visual utama yang menonjolkan estetika biji kopi berkualitas.
* **Informasi Produk:** Penjelasan mengenai kesegaran kopi harian (*Fresh Roast Daily*).
* **Call to Action (CTA):** Tombol interaktif untuk mengarahkan pengguna melakukan pemesanan atau kunjungan.

## 🛠️ Teknologi yang Digunakan

Proyek ini dibangun menggunakan teknologi *frontend* murni untuk performa yang ringan:

* **HTML5:** Untuk struktur konten web yang semantik.
* **CSS3:** Untuk styling, layouting (Flexbox/Grid), dan desain kartu yang modern.
* **JavaScript:** Untuk fungsionalitas interaktif dasar.

## 💻 Cara Menjalankan Proyek

1. *Clone* repositori ini:
```bash
git clone https://github.com/asboyy/Website-UMKM-KOPI-.git

```


2. Buka folder proyek.
3. Klik dua kali pada file `index.html` untuk melihat hasilnya di browser Anda.
